本頁概述 StorageGRID 如何支援 Amazon Simple Storage Service （ S3 ） API 。

本頁僅包含 StorageGRID 支援的 S3 作業。


NOTE: 若要查看每項作業的 AWS 文件、請選取標題中的連結。



== 通用 URI 查詢參數和要求標頭

除非另有說明、否則支援下列常見的 URI 查詢參數：

* `versionId` （視物件作業需求而定）


除非另有說明、否則支援下列常見的要求標頭：

* `Authorization`
* `Connection`
* `Content-Length`
* `Content-MD5`
* `Content-Type`
* `Date`
* `Expect`
* `Host`
* `x-amz-date`
