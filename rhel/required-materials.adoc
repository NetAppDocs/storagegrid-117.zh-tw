NetApp StorageGRID 產品授權
 a| 
您必須擁有有效且經過數位簽署的NetApp授權。

*附註*：StorageGRID 非正式作業授權、可用於測試及概念驗證網格、包含在《》安裝檔案中。

 a| 
此系統是透過維修用筆記型電腦來安裝。StorageGRID

* 網路連接埠
* SSH用戶端（例如Putty）
